# dotfiles

## Usage

Pull the repository, and then create the symbolic links using dotbot.

```bash
$ git clone git@github.com:Bowser1704/dotfiles.git ~/.dotfiles
$ cd ~/.dotfiles
$ ./install
```

- Neovim
- alacritty
- tmux
- rofi
- bspwm
